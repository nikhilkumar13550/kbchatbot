# Manulife KB ChatBot - Genpact AI Foundry Integration

This project integrates a **React (Vite) frontend** with a **FastAPI (Python) backend** to communicate with the Azure AI Foundry `KbChatBot` using the **Activity Protocol**.

## 🛠 Prerequisites
- Python 3.9+
- Node.js (v18+)
- Access to Genpact Azure AI Foundry (Entra ID)

## 📂 Project Structure
- `/frontend` - React + Vite application (Manulife UI)
- `/backend`  - FastAPI server (`main.py`)
- `requirements.txt` - Python dependencies

## 🚀 Getting Started

### 1. Backend Setup
Navigate to the root folder:
```bash
# Install dependencies
pip install -r requirements.txt

# Start the FastAPI server
python main.py




# Install dependencies
